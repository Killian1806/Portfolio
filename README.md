#  Mon Portfolio - Killian Mahut

Bienvenue sur le dépôt de mon portfolio personnel ! Ce site présente mon parcours, mes compétences techniques et les projets sur lesquels j'ai travaillé.

##  Stack Technique
* **Structure :** HTML5
* **Style :** CSS3 (Variables, Flexbox, Grids, Animations)
* **Interactivité :** JavaScript (ES6)
* **Contact :** Intégration API Web3Forms
* **Hébergement :** GitHub Pages

##  Fonctionnalités
* **Design Responsive :** Adapté aux mobiles, tablettes et ordinateurs.
* **Modales de Projets :** Système de cartes interactives avec descriptions tronquées et vues détaillées.
* **Navigation Fluide :** Menu avec indicateur de position active.
* **Conformité RGPD :** Mentions légales et politique de confidentialité intégrées.
